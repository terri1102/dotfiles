## To get started
1. Install packages
```bash
sudo apt update
```
```bash
sudo apt install -y zsh curl git fzf
```

2. Clone this repo
```
git clone https://github.com/terri1102/dotfiles.git
```

4. Run script
```
chmod +x setup_zsh.sh`
```
```
./setup_zsh.sh
```
